⚡ Quick Start
  1. Click ⚡ Ping All in the health bar to check service connectivity.
  2. Go to Auth → Register to create a test user.
  3. Go to Auth → Login — the JWT token auto-saves for all subsequent requests.
  4. Test the Product, Order, and Payment endpoints — the Bearer token is injected automatically.
