# Phase 2.3: Admin Oversight Portal - Context

## Overview
The Admin Oversight Portal is the internal command center for NexusMarket. It provides administrators with the tools necessary to monitor platform health, manage users, and moderate content.

## Core Requirements
1.  **Platform Statistics**: High-level view of users, products, and revenue.
2.  **User Management**: Ability to view all users and suspend/unsuspend accounts.
3.  **Product Moderation**: Oversight of all products, including the ability to unpublish items.
4.  **Order Feed**: Visibility into all transactions across the platform.

## Technical Constraints
- **Security**: Must enforce strict `ADMIN` role checks.
- **Microservices**: Data must be aggregated from Auth, Product, and Order services.

## Next Steps
- [ ] Implement user list and suspension in `auth-service`.
- [ ] Implement global order feed in `order-service`.
- [ ] Build the Admin UI shell and management pages in `apps/web`.
