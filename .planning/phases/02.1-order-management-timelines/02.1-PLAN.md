</read_first>
<action>
1. In `order.controller.ts`, implement `getVendorOrders`:
   - Query `Order` where `items.vendor === req.user._id`.
   - Return orders with relevant vendor items highlighted.
2. Implement `updateItemStatus`:
   - Takes `orderId`, `productId`, and `newStatus`.

</action>

</acceptance_criteria>
</task>
