## WebSocket Protocol Selection

| Option | Description | Selected |
|--------|-------------|----------|
| Socket.io | Rich features (reconnection, rooms, namespaces). Scales with Redis. | ✓ |
| Raw WebSockets | Lightweight, faster, but requires custom logic. | |

**User's choice:** Socket.io (Recommended)
**Notes:** User accepted the recommendation to use Socket.io for its robustness and ease of scaling in a microservices environment.

| Option | Description | Selected |

## Authentication

| Option | Description | Selected |
|--------|-------------|----------|

**User's choice:** Handshake JWT (Recommended)
**Notes:** Token will be passed during the connection handshake to ensure only authenticated users can connect.

| Option | Description | Selected |
