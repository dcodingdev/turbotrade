## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0a0a0a | Background (Dark mode primary) |
| Secondary (30%) | #171717 | Card background, Sidebars, Message bubbles (Received) |
| Accent (10%) | #7c3aed | Primary CTA, Unread indicators, Message bubbles (Sent) |
| Destructive | #ef4444 | Clear chat, Block user actions |

Accent reserved for: Primary CTA buttons, active conversation indicators, unread message badges, and the sender's own message bubbles.
